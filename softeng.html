Gift Tax Service

This was a group project for Software Engineering at University of Wisconsin - Whitewater.

The project was to make a web interface that would allow the user to fill out a form of gift tax information.

The web interface would then automatically fill out Form 709 as a PDF document and allow the user to download it.

I created the base template that we used for all of our HTML pages, and programmed the code that automatically filled out the form as a PDF document.

The project was coded in HTML, CSS, Javascript, and PHP.
