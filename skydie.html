Sky Die

Sky Die is a class project Unity game made for MAGD 488 at University of Wisconsin - Whitewater.

The game is a multiplayer versus game in which up to four players are falling through the sky, and must bump each other into obstacles in order to be the last one standing.

I was responsible for programming work in this project including handling game inputs, as well as working on character abilities and other gameplay features.

The game is available to play on Steam.
